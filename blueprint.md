
# F1 소개 사이트 청사진

## 개요

이 문서는 최신 웹 표준을 사용하여 현대적이고 반응이 빠르며 동적인 F1 소개 웹사이트를 제작하기 위한 계획을 설명합니다. 사용자는 탭 형식의 인터페이스를 통해 다양한 F1 관련 정보를 탐색할 수 있습니다.

## 디자인 및 기능

### 시각적 디자인

*   **레이아웃:** 데스크톱과 모바일 기기 모두에서 사용하기 쉬운 동적 탭 기반 인터페이스.
*   **색상 팔레트:** F1의 역동성을 나타내는 강렬한 기본 색상과 현대적인 보조 색상.
*   **타이포그래피:** 가독성과 시각적 계층 구조를 위한 명확하고 현대적인 글꼴 사용.
*   **상호작용성:** 활성화된 탭과 상호작용 요소에 시각적 피드백(예: "빛나는" 효과)을 제공합니다.

### 기능

*   **동적 콘텐츠 탭:** 메인 페이지는 F1에 대한 기본 소개를 표시합니다. 헤더의 탐색 버튼을 클릭하면 아래와 같은 콘텐츠가 동적으로 로드됩니다.
    *   **2026 정보:** 2026년 시즌의 팀 및 드라이버 정보 (가상 데이터 기반).
    *   **F1 용어:** F1에서 자주 사용되는 기술 및 레이싱 용어 목록과 설명.
    *   **뉴스:** **실제 최신 F1 소식**을 기반으로 한 뉴스 콘텐츠.
*   **웹 컴포넌트:** '드라이버 카드'와 같은 재사용 가능한 UI 요소는 웹 컴포넌트로 계속 관리하여 코드의 모듈성을 높입니다.

## 현재 계획

사이트 구조를 탭 기반의 동적 콘텐츠 시스템으로 재구성하고, 뉴스 섹션을 실제 소식 기반으로 구현합니다.

1.  **`index.html` 수정:** 헤더의 탐색 메뉴를 '2026 정보', 'F1 용어', '뉴스' 버튼으로 변경하고, 동적 콘텐츠가 표시될 메인 컨테이너 영역을 설정합니다. (완료)
2.  **`main.js` 재구성:**
    *   탐색 버튼에 클릭 이벤트 리스너를 추가합니다.
    *   각 버튼에 해당하는 콘텐츠(2026 정보, 용어, **실제 뉴스**)를 생성하고 메인 컨테이너에 렌더링하는 함수를 구현합니다.
    *   'F1 용어'와 **실제 뉴스**에 대한 새로운 데이터(객체 배열)를 추가합니다.
3.  **`style.css` 업데이트:** 새로운 탭 인터페이스와 뉴스 기사와 같은 동적 콘텐츠 영역의 스타일을 조정하고, 활성화된 탭의 스타일을 추가합니다.
